# Contributing

Thank you for your interest. At this time we are not accepting direct contributions.

- Current status: The project is still being shaped by the core maintainers and is not open for general pull requests or external governance.
- How to contribute now: If you would like to contribute, please contact one of the project maintainers to discuss involvement and next steps.
- Future plans: Once the project is ready, we will publish contribution guidelines and open the project to broader contributions and governance.
